## Instalação

A instalação é fácil, siga os passos abaixo :)

### Requisitos
```sh
* npm
```

### Instalação

1. Clone o Repositório
```sh
git clone https://github.com/brtec/node-discord-bot.git
```

2. Instale os pacotes NPM
```sh
npm install
```

2. Execute o script
```sh
node index.js
```
